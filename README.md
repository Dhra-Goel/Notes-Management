# Task Journal

A full-stack web application that helps users organize their daily tasks with priority levels and due dates. The application provides a clean interface for managing tasks while using a Node.js backend with MongoDB for persistent storage.

---

## Features

- Create new tasks
- Assign due dates
- Set task priority (Low, Medium, High)
- Mark tasks as completed
- Delete tasks
- Filter tasks by priority
- Responsive and modern user interface
- MongoDB database integration
- RESTful API using Express.js

---

## Installation

Clone the repository

```bash
git clone https://github.com/Dhra-Goel/Notes-Management.git
```

Move into the project folder

```bash
cd task-journal
```

Install dependencies

```bash
npm install
```

Start the server

```bash
node server.js
```

Open your browser

```
http://localhost:3000
```
